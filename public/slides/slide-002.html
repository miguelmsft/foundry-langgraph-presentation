What is Microsoft Foundry v2?

The unified AI platform. One resource. One SDK. One portal.

“Microsoft Foundry is a unified Azure platform-as-a-service for enterprise AI operations, model builders, and application development.”

  • 1,900+ models — OpenAI, DeepSeek, Meta, Mistral, Anthropic, and more
  • 1,400+ tools — MCP, A2A, function calling, code interpreter, web search
  • One project endpoint — https://<resource>.services.ai.azure.com/api/projects/<project>
  • One SDK — azure-ai-projects 2.x

Replaces the fragmented Azure AI Studio / Azure OpenAI / AI Services approach.

Microsoft Foundry — formerly Azure AI Foundry, formerly Azure AI Studio — is the new unified brand. Think of it as the single front door for all AI development on Azure. The key breakthrough is consolidation: previously you needed multiple packages, multiple endpoints, multiple portals. Now it's one resource type, one SDK, one portal at ai.azure.com. The model catalog has over 1,900 models and the tool catalog has over 1,400 tools. The new API surface is the Responses API, replacing the older Assistants API.
