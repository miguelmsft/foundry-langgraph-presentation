4 steps. Setup telemetry → Init data → Build agent → from_langgraph(agent).run()

This is the entire container entrypoint. Four steps: set up observability with Application Insights and the LangChain OpenTelemetry tracer, initialize the SQLite database from CSV files, build the LangGraph agent with the model, tools, and prompt, and then wrap it with from_langgraph() and call .run(). That last call starts an HTTP server on port 8088 that speaks Foundry's Responses API. This is the actual main.py from our demo — no simplification needed because it really is this simple.
