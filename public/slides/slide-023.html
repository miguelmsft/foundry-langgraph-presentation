🏭 Demo: Manufacturing Logistics Analyst Agent

An AI agent that autonomously analyzes inventory and shipment data, identifies critical issues, and produces actionable logistics reports.

Stack: LangGraph + gpt-5.4-mini + SQLite + Foundry Hosted Agent

Let's switch gears to the demo. We built a Manufacturing Logistics Analyst Agent — an AI agent that acts like a logistics analyst for a manufacturing facility. You give it access to inventory and shipment data, ask it a question, and it autonomously figures out what SQL queries to run, analyzes the results, and writes a complete logistics report. The entire thing runs as a Foundry Hosted Agent, so every decision is traced.
