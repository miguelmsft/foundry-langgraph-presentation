graph.py (simplified from our demo)
from langchain_openai import AzureChatOpenAI from azure.identity import DefaultAzureCredential, get_bearer_token_provider from langgraph.prebuilt import create_react_agent def _create_model() -> AzureChatOpenAI: token_provider = get_bearer_token_provider( DefaultAzureCredential(), "https://cognitiveservices.azure.com/.default", ) return AzureChatOpenAI( azure_endpoint=config.AZURE_OPENAI_ENDPOINT, azure_deployment=config.MODEL_DEPLOYMENT_NAME, azure_ad_token_provider=token_provider, temperature=0, streaming=True, ) def build_agent(): model = _create_model() tools = get_all_tools() # [list_tables, get_table_schema, query_database] return create_react_agent( model=model, tools=tools, prompt=SYSTEM_PROMPT, )

🔑 No API keys — DefaultAzureCredential + managed identity

Here's the actual code from our demo's graph.py. Three things to note. First: authentication uses DefaultAzureCredential with get_bearer_token_provider. This auto-refreshes tokens — critical for long-running hosted agents where tokens expire after an hour. No API keys anywhere. Second: temperature=0 makes the model deterministic. Third: create_react_agent is deceptively simple — one call that builds the entire ReAct state graph with the agent node, tool node, and conditional routing.
