This diagram shows how all the Python packages fit together. At the top is your code — the LangGraph agent, tools, and prompts. LangGraph provides the graph-based orchestration engine. langchain-openai provides the AzureChatOpenAI class. langchain-core provides the @tool decorator. The hosting adapter wraps everything into the Foundry-compatible HTTP server. At the bottom, azure-identity handles auth and azure-monitor-opentelemetry exports traces and metrics.
