The full picture. The container holds the LangGraph agent with its three SQLite tools and the hosting adapter. The agent calls gpt-5.4-mini deployed as GlobalStandard in the same Foundry account. The hosting adapter wraps everything into the HTTP server on port 8088. OpenTelemetry traces flow automatically to Application Insights and the Foundry Control Plane. The SQLite database is bundled inside the container image — built at Docker build time from CSV files.
