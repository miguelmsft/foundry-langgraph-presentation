What is LangGraph?

A framework for building AI agents as state graphs.

Key concepts:
Nodes = Functions
Edges = Transitions
State = Message history
Cycles = Agent loops

Not just chains — it's a state machine with cycles, conditions, and persistence.

LangGraph is LangChain's low-level orchestration framework. The key mental model is a graph: nodes are functions, edges are transitions. Unlike simple LLM chains that go A → B → C, LangGraph supports cycles — the agent can loop back, retry, iterate. It's at version 1.1.6, released in April 2026, and is used in production by companies like Klarna, Replit, and Elastic. The key difference from LangChain proper: LangChain is high-level composable abstractions, LangGraph is low-level full-control state graphs.
